## Results

- **17% increase** in challenger user cohort page views
- **13% increase** for all users 30 days after production release for 100% of users

## My Impact

- Conducted discovery research including user interviews and surveys that validated the concept and generated sufficient momentum for advancement.
- Secured stakeholder approval for this initiative, which later validated a broader personalization program.

## The Problem

Media platforms struggle against ultra-personalized competitors like X, YouTube, TikTok, and Instagram due to insufficient content recommendations. The challenge was to validate whether investment in machine learning or complex recommendation logic is justified without massive upfront costs.

![The problem](/images/projects/the-australian/problem.svg)

## The Solution

With the smallest possible investment, validating if the proposed program actually delivers value at a smaller scale so that we can then scale up. Leveraging third-party algorithms to test if content personalization generates measurable value before committing to enterprise-scale solutions.

![The solution](/images/projects/the-australian/solution.svg)

## Key Learning

When presenting to busy executives, maintain simplicity and sharp messaging. Focus on customer and audience benefits rather than complexity.
